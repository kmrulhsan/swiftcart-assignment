1) What is the difference between null and undefined?
    ans: undefined means a value is declared but not assigned and null means declared as nothing by the developer

2) What is the use of the map() function in JavaScript? How is it different from forEach()?
    ans: These two functions are used as loops but map function returns a new modified array and foreach returns an undefined which is used to perform side effects like api calling.

3) What is the difference between == and ===?
    ans: == uses type coersion to check the values basically convert the values before equalling and === checks both value and data types are equal or not. It does not do any kind of conversion.

4) What is the significance of async/await in fetching API data?
    ans: async/await is used for making asynchornous code to behave like a synchronous code
    
5) Explain the concept of Scope in JavaScript (Global, Function, Block).
    ans:  scopes in JS determines where values are accessible in the code. In global scope variable declared outside of the block, In function scope variables are declared inside the function, block scope is any kind of inside curly braces{}.
